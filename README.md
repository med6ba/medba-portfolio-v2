## Installation

1. Clone the repo:
```bash
git clone https://www.github.com/med6ba/medba-portfolio-v2.git
```
2. Open the project folder in your code editor.
3. Install dependencies:
```bash
npm install
```
4. Start dev server:
```bash
npm run dev
```
5. Lunch the website:
```bash
http://localhost:8080
```
